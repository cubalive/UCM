A) Dialysis Load Predictor (V1 = “IA ligera real”)
- Implement forecast for each clinic for next 180 minutes in 15-min buckets.
- Model V1 (no external ML infra):
  1) Build historical baseline from completed/scheduled trips grouped by:
     - day_of_week
     - time_bucket_15m
     - service_level (ambulatory/wheelchair)
     - inbound vs outbound
  2) Use rolling adjustment from last 7 days (recent drift).
  3) Output forecast with confidence:
     - LOW if low history count
     - MEDIUM if >= N samples
     - HIGH if >= M samples
- Create endpoint or RPC:
  - get_clinic_forecast(clinic_id, horizon_minutes=180, bucket=15)
  returns [{bucket_start, inbound_amb, inbound_wc, outbound_amb, outbound_wc, confidence}]

B) Capacity Forecast (drivers needed)
- Convert forecast to drivers required:
  - Define capacity_per_driver per service_level (config table):
    wheelchair: avg_cycle_minutes_wc
    ambulatory: avg_cycle_minutes_amb

- Output:
  - “Need +X wheelchair drivers 7–10am”
  - “Shortage risk” if available drivers < needed
- Create endpoint/RPC:
  - get_clinic_capacity_forecast(clinic_id) -> includes shortages + recommended drivers
