- ARRIVED_PICKUP
- PICKED_UP
- EN_ROUTE_TO_DROPOFF
- COMPLETED

Driver UI buttons on trip:

Backend:
- POST /api/trips/:id/status { status }

- Must follow valid transitions (no skipping)
- Record timestamps:
  started_at, arrived_pickup_at, picked_up_at, en_route_dropoff_at, completed_at
