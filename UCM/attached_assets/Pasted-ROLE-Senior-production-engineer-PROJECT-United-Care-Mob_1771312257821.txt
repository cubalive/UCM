PHASE 1 — DB (ADD ONLY)
A) Extend invoices:
- invoice_number TEXT UNIQUE NULL (INV-000001 generated at finalize)
- payment_status TEXT CHECK IN ('unpaid','partial','paid','overdue') NOT NULL DEFAULT 'unpaid'
- amount_paid_cents BIGINT NOT NULL DEFAULT 0
- balance_due_cents BIGINT NOT NULL DEFAULT 0
- due_date TIMESTAMPTZ NULL
- stripe_checkout_session_id TEXT NULL
- stripe_payment_intent_id TEXT NULL
- stripe_checkout_url TEXT NULL
- last_payment_at TIMESTAMPTZ NULL

B) Create invoice_payments:
- id UUID PK default gen_random_uuid()
- invoice_id UUID FK invoices.id ON DELETE CASCADE
- amount_cents BIGINT NOT NULL
- method TEXT CHECK IN ('stripe','ach','manual') NOT NULL
- reference TEXT NULL
- stripe_payment_intent_id TEXT NULL
- stripe_charge_id TEXT NULL
- paid_at TIMESTAMPTZ NOT NULL DEFAULT now()
- created_at TIMESTAMPTZ NOT NULL DEFAULT now()
Index: invoice_payments(invoice_id, paid_at desc)
Idempotency: unique on (stripe_payment_intent_id) where not null OR (reference) for session id.

PHASE 2 — SERVER LOGIC
A) On invoice FINALIZE:
1) Generate invoice_number atomically (INV-000001…):
   - Create invoice_sequences table with a single row counter, or DB sequence.
2) Set due_date = period_end + grace_days (from clinic_billing_settings; default 0)
3) Set payment_status='unpaid', amount_paid_cents=0, balance_due_cents=total_cents

B) Create single function updateInvoicePaymentStatus(invoiceId):
- sumPayments = SUM(invoice_payments.amount_cents)
- amount_paid_cents = sumPayments
- balance_due_cents = max(total_cents - sumPayments, 0)
- if balance_due_cents == 0 => paid
- else if amount_paid_cents > 0 => partial
- else unpaid
- if now > due_date and not paid => overdue
- last_payment_at = max(paid_at)
