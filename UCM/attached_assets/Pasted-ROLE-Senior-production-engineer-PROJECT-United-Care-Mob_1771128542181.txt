DECISION CONFIRMED: FULLY AUTOMATED CLINIC BILLING (NO MANUAL)
We will automatically bill every clinic trip leg using editable tariffs based on:
- passenger_count (1..4)
- leg_type (outbound/return)
- outcome (completed/no_show/cancelled/company_error)
- cancel timing window (advance vs same-day vs late)

Invoices must be transparent and grouped:
Clinic -> Week -> Day -> Patient -> Outbound then Return (one under the other)

============================================================
NON-NEGOTIABLE RULES
============================================================
1) Only clinic trips included (exclude private/internal UCM Direct).
2) Every trip leg gets a BILLING OUTCOME with reason + audit.
3) Cancel/No-show fees must respect time windows (editable).
4) Company Error default fee = $0 (editable).
5) Invoice lines must store SNAPSHOT rates so past invoices never change.
6) Returns included and displayed under outbound; legs billed independently by outcome.

PHASE 3 — CANCEL WINDOW CLASSIFICATION
============================================================
Define cancel_window ONLY when outcome=cancelled:
- advance: cancelled >= X hours before scheduled pickup
- same_day: cancelled < X hours but before pickup time
- late: cancelled after pickup time (or within Y minutes of pickup)

Make thresholds editable per clinic (profile-level settings) OR global defaults:
- cancel_advance_hours (default 24)
- cancel_late_minutes (default 0 or 15)
