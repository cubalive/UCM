PHASE 3 — B2 SMART PROMPTS (non-spam, deterministic)
================================================================================
3.1 Create smartPrompts engine:
Inputs:
- now, trip.status, scheduled_pickup_at, pickup coords, driver latest coords, ETA
Rules (configurable):
- LEAVE_NOW: if pickup in <= 25min and driver not en_route yet, once per trip
- ARRIVE_NOW: if within geofence 150m and still not marked arrived, once per trip
- LATE_RISK: if ETA exceeds (scheduled + grace), cooldown 10min
Anti-spam:
- store fired prompts per trip in local storage or driver_settings cache
- never repeat same prompt in same phase
