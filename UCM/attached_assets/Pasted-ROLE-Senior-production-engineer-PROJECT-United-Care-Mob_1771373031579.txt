- company_id uuid pk
- enabled boolean null  -- null means inherit global
- fee_type text null
- fee_percent numeric null
- fee_cents int null
- updated_at timestamptz default now()

C) Add columns to billing invoices (or invoicePayments) to store the fee used at charge time:
- platform_fee_cents int not null default 0

- platform_fee_rate numeric null

A) Implement resolver getEffectivePlatformFee(companyId):
- Read global settings row id=1
- Read company override row (if any)

  - fee_type = company.fee_type ?? global.default_fee_type

  - Keep transfer_data.destination as company connected account
- Store fee details on invoice BEFORE redirecting (platform_fee_cents/type/rate/net_to_company)
