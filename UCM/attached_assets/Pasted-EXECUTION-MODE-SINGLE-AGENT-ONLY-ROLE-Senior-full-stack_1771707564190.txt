- Loads trip pickup/dropoff lat/lng from DB
- If trip.route_polyline is present AND pickup/dropoff unchanged → return existing route fields
- Else call Google Directions API:
    GET https://maps.googleapis.com/maps/api/directions/json
      ?origin={pickupLat},{pickupLng}
      &destination={dropoffLat},{dropoffLng}
      &mode=driving
      &key={GOOGLE_MAPS_API_KEY}
- Validate response:
    - if no routes[0] → throw typed error “NO_ROUTE_FOUND”
- Extract:
    - encoded polyline: routes[0].overview_polyline.points
    - distance meters: routes[0].legs[0].distance.value
    - duration seconds: routes[0].legs[0].duration.value
- Store into trips row:
    route_polyline, route_distance_meters, route_duration_seconds
- Return the stored values.

Implement with existing HTTP client already used in repo (fetch/axios). Do not add heavy dependencies.
