PHASE 2 — BILLING COMPUTE ENGINE (DETERMINISTIC + IDEMPOTENT)
========================================================
Implement computeTripBilling(tripId):
- Load trip and determine if shared:
  - if shared_group_id exists: load all legs (trips) in that group
  - else: one leg

For EACH billed leg:
- billed patient_id = leg.patient_id
- billed clinic_id  = leg.clinic_id (must exist or mark pricing_missing in components; do NOT invoice)
- Determine service_date from scheduled_pickup_at date (or trip date field)

PRICING PRIORITY (v1 production-safe):
1) If status_at_bill = NO_SHOW: total = no_show_fee_cents (from effective tariff)
2) Else if CANCELLED: total = cancel_fee_cents
3) Else if recurring schedule pricing_mode=CONTRACT and contract_price_cents set:
   total = contract_price_cents; pricing_mode='CONTRACT'
4) Else:
   TARIFF (base + miles*per_mile + minutes*per_minute + wheelchair surcharge; enforce minimum fare)

Shared handling:
- PER_PATIENT: each leg uses its own computed total
- SPLIT:
   - Compute route_total ONCE (use primary_trip_id if present else first leg)
   - Split cents without losing pennies:
       split = floor(route_total / passenger_count)
       remainder = route_total - split*passenger_count
       first leg gets split + remainder, others get split
   - pricing_mode='SPLIT'
   - components must show split math

Snapshot rules:
- Always write/update trip_billing with unique(trip_id, patient_id) (upsert).
- components MUST include:
  - tariff_snapshot (all tariff fields used)
  - miles/minutes source + missing flags
  - wheelchair/passengers/shared flags
  - formulas + final numbers
  - status_at_bill + reasons (cancel/no_show)

Trigger points:
- When trip enters finalized state (COMPLETED/CANCELLED/NO_SHOW): compute + upsert.
- Provide backfill endpoint:
POST /api/company/billing/backfill?from=YYYY-MM-DD&to=YYYY-MM-DD&city_id=...
Auth: SUPER_ADMIN or DISPATCH/COMPANY scoped
- Computes trip_billing for any finalized trips missing billing rows.
