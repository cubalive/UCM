# Clinic App Icons

Replace these placeholder files with production-ready icons before store submission.

## Required Files

### iOS (`ios/`)
- `AppIcon-1024.png` — 1024x1024, no transparency, no rounded corners (iOS adds rounding automatically)

### Android (`android/`)
- `AppIcon-512.png` — 512x512, used as the Play Store listing icon
- `ic_launcher_foreground.png` — 512x512 with safe zone (inner 66%), adaptive icon foreground layer
- `ic_launcher_background.png` — 512x512, adaptive icon background layer (solid color or pattern)

### PWA (`pwa/`)
- `icon-192.png` — 192x192, standard PWA icon
- `icon-512.png` — 512x512, standard PWA icon
- `maskable-192.png` — 192x192, maskable (safe zone inner 80%)
- `maskable-512.png` — 512x512, maskable (safe zone inner 80%)
