STEP 2 — BACKEND: SERVICE TO COMPUTE & STORE ROUTE POLYLINE
Implement server function/service: ensureTripRoute(tripId)
- Load trip pickup/dropoff lat/lng (and tenant/company scope enforced).
- Compute fingerprint = `${pickupLat},${pickupLng}|${dropoffLat},${dropoffLng}`
- If trip.route_polyline exists AND trip.route_fingerprint === fingerprint → return existing route fields without calling Directions.
- Add in-process “single-flight” lock to prevent duplicate Directions calls for same trip concurrently:
  - Map<tripId, Promise> inFlightRoutes
  - If inFlightRoutes has tripId → await it and return.
- Else call Google Directions API (backend only):
  GET https://maps.googleapis.com/maps/api/directions/json
    ?origin={pickupLat},{pickupLng}
    &destination={dropoffLat},{dropoffLng}
    &mode=driving
    &key={GOOGLE_MAPS_API_KEY}
- Validate response:
  - if !routes[0] → throw typed error “NO_ROUTE_FOUND”
- Extract:
  - encoded polyline: routes[0].overview_polyline.points
  - distance meters: routes[0].legs[0].distance.value
  - duration seconds: routes[0].legs[0].duration.value
- Store into trips row:
  route_polyline, route_distance_meters, route_duration_seconds, route_fingerprint
- Return stored values.
Implement using existing HTTP client (fetch/axios) already in repo. No heavy deps.

Add:
GET /api/trips/:id/route
- Auth required
- Verify tenant/company access using existing RBAC/tenant helpers (404 if not accessible)
- Calls ensureTripRoute(tripId)
- Returns JSON:

- Do not fail the main request if Directions fails.

- GET /route can retry later.

STEP 4 — FRONTEND: DRAW POLYLINE INSTEAD OF STRAIGHT LINE
In all trip map views (admin trip details, driver trip screen, clinic view):
- Replace current line from [pickup, dropoff] with decoded polyline from /api/trips/:id/route.
- Fetch once when opening trip screen.
- Decode:
  const path = google.maps.geometry.encoding.decodePath(routePolyline)
- Render:
  new google.maps.Polyline({ path, strokeOpacity: 0.9, strokeWeight: 5 }).setMap(map)
- Ensure Maps loader includes geometry library:
  - If script tag: &libraries=geometry
  - If JS loader: libraries: ["geometry"]
- If routePolyline missing (API failure), fall back to straight line so UI never breaks.
