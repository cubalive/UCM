DEFINITIONS (single source of truth)
Trip phases (must map to visual + availability):
A) SCHEDULED / ASSIGNED (future trip, not started)
B) EN_ROUTE_TO_PICKUP
C) ARRIVED_PICKUP (geofence or driver confirm)
D) PICKED_UP / EN_ROUTE_TO_DROPOFF
E) ARRIVED_DROPOFF
F) COMPLETED
G) CANCELLED / NO_SHOW

Driver availability:
A driver is AVAILABLE for assignment only if:
- on_shift = true
- last_seen_at within ACTIVE_WINDOW_MINUTES (default 5)
- NOT currently engaged in an active trip whose status is in:
  { EN_ROUTE_TO_PICKUP, ARRIVED_PICKUP, PICKED_UP, EN_ROUTE_TO_DROPOFF, ARRIVED_DROPOFF }
- gps_status='ENABLED'
- is_test_driver=false (unless SUPER_ADMIN includeTest)
