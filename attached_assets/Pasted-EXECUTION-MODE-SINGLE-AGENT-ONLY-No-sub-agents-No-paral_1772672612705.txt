STEP 3 — Install image generation tooling

Install dev dependency:

npm install sharp --save-dev

STEP 4 — Create icon generator script

Create file:

scripts/generate-icons.mjs

The script must:

- Read each SVG
- Generate PNG icons in sizes:
1024
512
192

Output to:

client/public/generated-icons/driver
client/public/generated-icons/clinic
client/public/generated-icons/admin

Filenames:

icon-1024.png
icon-512.png
icon-192.png

STEP 5 — Add package.json scripts

Add:

"icons:driver": "node scripts/generate-icons.mjs driver",
"icons:clinic": "node scripts/generate-icons.mjs clinic",
"icons:admin": "node scripts/generate-icons.mjs admin",
"icons:all": "node scripts/generate-icons.mjs all"

STEP 6 — Generate icons automatically

Run:

npm run icons:all
