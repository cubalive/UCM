============================================================

Create tables:

intelligence_publications
- id uuid pk
- module text  -- 'indexes'|'certification'|'ranking'|'audit'|'prediction'
- quarter_key text nullable
- scope text nullable  -- 'general'|'state'|'city'|'clinic'
- state text nullable

- enabled bool default true
UNIQUE(publication_id, target_type, clinic_id)

- Create publication (choose module, quarter, scope, targets)

- Publish / Unpublish (revokes instantly)
- Audit log of publish actions (who/when)

CONFIG JSON (controls “how I publish”):
config_json supports:
- show_full_ranking_list: boolean (default false)
- show_peer_names: boolean (default false; anonymize others)
- show_counts: boolean (default true)
- show_thresholds: boolean (default true)
- show_trends: boolean (default true)
- allow_pdf_download: boolean (default true)
- visible_metrics: array of metric_keys (optional whitelist)
- notes: text (optional message shown to clinic)
